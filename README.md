# vue-easily
* 该项目仅仅是为了演示vue，主旨简单易懂的学习使用vue
* 涉及到webpack,vue-router,vue-resource,vuex等

## 使用
* 生产环境可将 nginx 等配置到web目录并重写路由——非资源映射到index.html
* development 环境会开启一个api服务

``` bash
$ npm install #插入依赖
$ npm run dev #编译并监视文件改动时热加载
$ npm run build #编译文件到指定目录
```
